# Relay Drawler

Prototype web mobile d'un roguelite de drones auto-replicateurs.

Le jeu est concu pour etre joue en portrait, d'une main, sans temps reel. Le joueur lance des expeditions, tire des modules, garde et relance une selection, gere chaleur et reseau, puis choisit entre securiser son loot ou pousser plus loin la run.

## Lancer le prototype

Ouverture directe :

```text
index.html
```

Serveur local :

```bash
python3 -m http.server 4173
```

Puis ouvrir :

```text
http://127.0.0.1:4173
```
